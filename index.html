Open source video game library manager

One simple goal. To provide a unified interface for all of your games.

What is Playnite?

An open source video game library manager and launcher with support for 3rd party libraries like Steam, GOG, Origin, Battle.net and Ubisoft Connect. Includes game emulation support, providing one unified interface for your games.

Fullscreen mode

Playnite offers desktop and fullscreen mode. Both of them support custom user themes.

Other features

Fast

Playnite's user interface is fast and light on resources, even with large libraries.

Free

No Playnite features are locked behind a paywall and the complete source code is available under the MIT license.
